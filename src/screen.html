2D Web Based CAD

Aplikasi web ini merupakan Computer Aid Design (CAD) berbasis WebGL. Berikut adalah fitur-fitur yang disediakan oleh website.

Cursor

Dengan Cursor, Anda memiliki 2 mode:

Recolor Mode

Bangun yang ada di canvas dapat diwarnai ulang dalam recolor mode.

  1. Pilih warna dari tombol Color
  2. Klik bangun yang ingin diubah warnanya

Resize Mode

Gambar yang ada di canvas dapat diubah ukurannya ulang dalam recolor mode.

  1. Pilih warna dari tombol Color
  2. Tentukan bangun yang ingin diubah ukurannya
  3. Klik dan tahan salah satu titik dari bangun
  4. Tarik titik sudut bangun, kemudian lepaskan untuk mengubah ukurannya

Line

Dengan Line, Anda dapat membuat sebuah garis.

  1. Pilih tab 'Line'
  2. Pilih warna dari tombol Color
  3. Klik di kanvas untuk menentukan titik awal garis
  4. Klik lagi di kanvas untuk menentukan titik akhir garis
  5. Tekan escape selama menentukan titik kedua untuk membatalkan penggambaran

Square

Dengan Square, Anda dapat membuat sebuah persegi.

  1. Pilih tab 'Square'
  2. Pilih warna dari tombol Color
  3. Klik di kanvas untuk menentukan sudut awal persegi
  4. Klik lagi di kanvas untuk menentukan titik sudut persegi yang berlawanan
  5. Tekan escape selama menentukan titik kedua untuk membatalkan penggambaran

Polygon

Dengan Polygon, Anda dapat membuat sebuah poligon dengan jumlah sisi banyak. Jumlah sisi tergantung dari berapa kali Anda menentukan titiknya.

  1. Pilih tab 'Polygon'
  2. Pilih warna dari tombol Color
  3. Klik di kanvas untuk menentukan titik awal poligon
  4. Klik lagi di kanvas untuk menentukan titik selanjutnya
  5. Titik dari poligon tidak terbatas dan dapat dibuat sebanyak mungkin
  6. Untuk mengakhiri gambar, klik titik awal dari poligon
  7. Tekan escape selama menentukan titik kedua untuk membatalkan penggambaran

Eyedropper

Gunakan tombol Eyedropper untuk menyalin warna yang terdapat di layar untuk menentukan warna dari bangun yang akan digambar.

Color

Gunakan tombol Color untuk memilih warna untuk menentukan warna dari bangun yang akan digambar.

Clear

Gunakan tombol Clear untuk menghapus semua bangun yang digambarkan di kanvas.

Save

Gunakan tombol Save untuk menyimpan hasil gambar ke dalam sebuah file json.

Browse

Gunakan tombol Browse untuk membuat file json yang pernah disimpan sebelumnya
